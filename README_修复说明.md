# 梦之星邀请函 H5：版式修复说明

本版本修复了预览时出现的：

1. 文字左右被裁切
2. 图片左右被裁切
3. “更多闪光瞬间”等照片页标题偏移
4. 桌面预览器中 430px 手机壳与 vw 宽度冲突的问题

## 修复方式

- 将原先部分 `92vw` 宽度改为相对页面容器的 `calc(100% - xxpx)`。
- 给活动详情、信件正文、字幕等文本增加自动换行保护。
- 轮播图与九宫格图片改为 `object-fit: contain`，尽量完整显示原图。
- 给完整显示的图片后方增加同图模糊背景，避免出现生硬黑边。
- 修复绝对定位标题的 `translateX(-50%)` 被入场动画覆盖的问题。

## 音乐文件

如果你使用这个压缩包重新部署，请重新确认音乐文件仍位于：

```text
assets/audio/bgm.mp3
```

文件名必须是 `bgm.mp3`，否则页面会找不到音乐。浏览器当然不会主动说自己找不到，毕竟错误提示太有用了，不符合人类互联网传统。
