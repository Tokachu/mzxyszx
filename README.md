# 宿松县梦之星艺术中心成人声乐班十周年邀请函 H5

这是一个可直接部署的静态 H5 电子邀请函。核心文件是：

- `index.html`：页面结构和文字
- `css/style.css`：动效、红金封面、信封动画、照片轮播、金色字效
- `js/app.js`：音乐控制、上划页、星光粒子、轮播逻辑
- `assets/photos/`：照片素材
- `assets/audio/bgm.mp3`：背景音乐文件，请自行放入

## 使用方法

1. 解压整个文件夹。
2. 将合法获得/已授权使用的《我们的生活充满阳光》合唱版音频放到 `assets/audio/` 目录下，命名为 `bgm.mp3`。
3. 打开 `index.html`，搜索下面这些文字并修改：
   - `待定`：活动地点、演出时间
   - `2026 年　月　日`：落款日期
4. 将整个文件夹上传到任意静态网站托管服务，例如：
   - 腾讯云 COS 静态网站
   - 阿里云 OSS 静态网站
   - Cloudflare Pages
   - Vercel
   - Netlify
   - GitHub Pages
5. 获得 HTTPS 链接后复制到微信即可转发。

## 微信播放说明

微信内置浏览器一般不允许网页一打开就自动播放音乐，所以页面做了“点击开启邀请函”的入口。用户点击后会尝试播放背景音乐，然后自动进入信封页。人类平台限制，机器只能优雅绕行，不能当场掀桌。

## 后续可改内容

- 要换照片：替换 `assets/photos/` 中同名图片即可。
- 要改字幕：修改 `index.html` 中 `<figcaption>` 内的文字。
- 要加/删页面：复制一个 `<section class="page">...</section>` 模块即可。
- 要改颜色和动画：主要在 `css/style.css`。
